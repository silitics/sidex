# User Guide

This user guide will teach you what you need to know to successfully use Sidex with officially supported languages and interchange formats. It does not go into deep technical details but instead focuses on concrete use cases and applications.

Before reading this user guide, please follow the [getting started](../getting-started.md) instructions.

Please [open an issue](https://github.com/silitics/sidex/issues) if there is anything you feel is difficult to understand or can otherwise be improved.
