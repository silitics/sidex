# Sidex: Helpers for Converting Attributes

This crate is part of [Sidex](https://oss.silitics.com/sidex/). It contains functionality for converting Sidex IR attributes to Rust structures.

For examples, have a look at [`sidex-attrs-rust`](https://docs.rs/sidex-attrs-rust/) and [`sidex-attrs-json`](https://docs.rs/sidex-attrs-json/).
